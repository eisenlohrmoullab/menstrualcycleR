# Example longitudinal daily dataset with cycle measures

This dataset contains daily symptom and cycle information used to test
and demonstrate the package.

## Usage

``` r
cycledata
```

## Format

A data frame with 619 rows and 5 variables:

- id:

  Participant ID

- daterated:

  Date of symptom observation

- symptom:

  Daily rating of symptom outcome of interest

- menses:

  `1` indicating first day of menstruation, menses onset

- ovtoday:

  `1` indicating estimated day of ovulation in study, based on
  biomarkers of ovulation.

## Source

Simulated or anonymized data
